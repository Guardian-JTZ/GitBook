## 谷歌语法

| 关键字        | 说明                                           |
| ---------- | -------------------------------------------- |
| site       | 指定域名                                         |
| intext     | 正文中存在关键字的网页                                  |
| intitle    | 标题中存在关键字的网页                                  |
| allintitle | 用法和intitle类似，只不过可以指定多个词，例如:alltitle:后台登录 管理员 |
| info       | 一些基本信息                                       |
| inurl      | URL存在关键在的网页                                  |
| allinurl   | 用法和inurl类似，只不过可以指定多个词例如：inurl:Login admin    |
| filetype   | 搜索指定文件类型                                     |
| cache      | 搜索已删除的缓存网页，推荐组合使用                            |
