## 参数说明

```shell
swaks --to michael --from JTZ --header "Subject: Testing!" --body "ignore this message" --server 10.10.11.166
--to  		# 收件人
--from 		# 要显示的发件人邮箱
--ehlo 		# 伪造的邮件 ehlo 头
--body 		# 邮件正文
--header 	# 邮件头信息，subject 为邮件标题
--data 		# 源邮件
--server 	# 邮件接收服务器地址
```
